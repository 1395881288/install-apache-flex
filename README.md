install-apache-flex
===================

Install Apache Flex SDK use AIR

### 在Apache Flex SDK Installer 4.8.0的基础上增加如下内容：  
* 自动下载Apache Flex SDK 4.9.0（原版下载的版本为：4.8.0）
* 自动下载Adobe AIR SDK 3.5.0（原版下载的版本为：3.1.0）

### 使用方法：  
直接下载/安装bin-release里面到air安装包即可。
